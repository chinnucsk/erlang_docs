1 Toolbar Release Notes

This document describes the changes made to the Toolbar application.

1.1  Toolbar 1.4.1

Improvements and New Features

  • The documentation is now built with open source tools (xsltproc and fop) that exists on most platforms. One visible change is that the frames are removed.

    Own Id: OTP-8201

1.2  Toolbar 1.4

Improvements and New Features

  • The copyright notices have been updated.

    Own Id: OTP-7851

1.3  Toolbar 1.3.0.1

Improvements and New Features

  • Minor Makefile changes.

    Own Id: OTP-6689

  • Obsolete guard tests (such as list()) have been replaced with the modern guard tests (such as is_list()).

    Own Id: OTP-6725

1.4  Toolbar 1.3

Improvements and New Features

  • Notification windows are now centered over a parent window.

    Own Id: OTP-6013 Aux Id: OTP-5987

1.5  Toolbar 1.2.1

Fixed Bugs and Malfunctions

  • The graphic applications now search for HTML documentation in the correct place.

    Own Id: OTP-5381
